GradioGround는 웹 기반의 [그라디오](https://www.gradio.app) 플레이그라운드입니다.
웹 브라우저에서 그라디오 앱을 작성하고, 구현 결과를 실시간으로 확인할 수 있습니다.


## 시작하기

다음 명령어를 입력하여 개발 서버를 실행합니다.

```bash
yarn dev
```

[http://localhost:3000](http://localhost:3000)에 접속하여 실행 결과를 확인합니다.
